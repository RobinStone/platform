1. Создаём папку с названием инструмента (приложения)2. Бросаем в неё картинку или SVG обязательно с именем icon (расширение может быть любым, лишь бы графика)3. Создаём файл about.txt (для отображения подсказки)4. Создаём файл ajax.php (сюда будут направляться все ajax запросы общего JS метода - SENDER_APP)5. Создаём файл controller.php (тут всё стандартно, вначале обрабатывается он а потом и сам шаблон приложения из файла template.php)6. Создаём файл template.php (тут HTML разметка приложения)7. Создаём файл main.js8. Создаём файл main.cssПри открытии админ-панели происходит авто-инициализация приложения и оно помещается в таблицу main "app_ИМЯприложения"Все основные настройки самого модуля APP, расположены как и положено в папках./CSS/apps.css и./JS/apps.js    для общей работы с классом app на JS существует свой класс
